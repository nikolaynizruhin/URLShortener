URL Shortener

Share your shortened URLs

Requirements:
- PHP 5.6+
- Apache 2.4 / Nginx
- PDO MySQL

Installation:
1. $ git clone https://github.com/nikolaynizruhin/URLShortener.git
2. $ composer install
3. Update the database info in include/config.php
4. Run the SQL included in url_shortener.sql
